📈 Stock Price Prediction using Machine Learning
📌 Project Overview
This project focuses on predicting stock prices using historical market data and machine learning models. The model was implemented in Google Colab, leveraging Python libraries such as NumPy, Pandas, and Scikit-learn.

🔍 Data Processing
Data Source: Stock market data was obtained from Yahoo Finance or Kaggle.
Preprocessing Steps:
Handling missing values.
Feature scaling (MinMaxScaler / StandardScaler).
Feature engineering (e.g., moving averages, volatility indicators).
🧠 Machine Learning Models Used
Linear Regression: A simple baseline model for price prediction.
Decision Trees: Captures non-linear patterns in stock price trends.
LSTM (Long Short-Term Memory): A deep learning approach for time series forecasting.
